# Odoo Custom Reports Portfolio

This repository showcases four custom Odoo reports developed for internal financial and operational management.  
Each report was built with a focus on accuracy, usability, and integration with core Odoo modules.

## Included Reports

1. **Budget Report**  
   Tracks planned vs. actual expenses, supports departmental analysis, and integrates with the accounting module for real-time insights.

2. **Caisse Management Report**  
   Provides daily and monthly summaries of cash transactions, teller balances, and reconciliation support across multiple branches.

3. **Contracts Report**  
   Summarizes active employee or client contracts, renewal timelines, and related financial commitments.

4. **Payroll Listings**  
   Generates detailed payroll summaries per period, including allowances, deductions, and net pay breakdowns.

## Technical Stack
- Odoo 15/16 (depending on implementation)
- Python
- PostgreSQL
- QWeb / XLSX report engine
